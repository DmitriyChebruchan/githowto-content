<h3>Metas</h3>

<ul><li>Fazer mudanças no repositório original para que possamos tentar receber essas mudanças.</li></ul>

<h2><em>01</em>Faça uma mudança no repositório original <strong>hello</strong></h2>

<h4 class="h4-pre">Execute:</h4>

<pre class="instructions">cd ../hello
# (You should be in the original hello repository now)</pre>

<p style="color:red;"><strong><span class="caps">NOTA</span>: Nós estamos agora no repositório <em>hello</em></strong></p>

<p>Faça as seguintes mudanças no arquivo <span class="caps">README</span>:</p>

<h4 class="h4-pre">Arquivo: <em><span class="caps">README</span></em></h4>

<pre class="file">This is the Hello World example from the git tutorial.
(changed in original)</pre>

<p>Agora adicione e faça commit dessas mudanças</p>

<h4 class="h4-pre">Execute:</h4>

<pre class="instructions">git add README
git commit -m "Changed README in original repo"</pre>

<h2><em>02</em> Próximo</h2>

<p>Agora o repositório original tem mudanças mais recentes que não estão incluídas na versão clonada. Em seguida, vamos receber essas mudanças no repositório clonado.</p>
